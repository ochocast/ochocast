# Usage

Welcome to the OchoCast user guide!

## What is OchoCast?

OchoCast is a multi-channel streaming application for live and on-demand events. It consists of two main technical parts:

- The "video storage" side, which manages, hosts and serves recorded content.
- The "live streaming" side, which supports launching live streams.
