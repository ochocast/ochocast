# Usage

Welcome to the OchoCast usage guide!

## What is OchoCast?

Ochocast is a multi-channel streaming application dedicated to live or recorded events. It consists of two main technical components:

- The "video storage" part, which allows you to manage, host, and distribute recorded content.
- The "live streaming" part, which allows you to launch live broadcasts.
